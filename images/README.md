# Images Folder Structure

This folder contains all images organized by page for easy access and management.

## Folder Structure

### Main Pages
- `home/` - Images for the main/home page (index.html)
- `about/` - Images for the About page
- `services/` - Images for the Services main page
- `industries/` - Images for the Industries main page
- `contact/` - Images for the Contact page
- `careers/` - Images for the Careers page
- `appointment/` - Images for the Appointment page

### Service Pages
- `services/financialreporting/` - Images for Financial Reporting service page
- `services/audit/` - Images for Audit service page
- `services/tax/` - Images for Tax service page
- `services/bookkeeping/` - Images for Bookkeeping service page
- `services/advisory/` - Images for Advisory service page
- `services/integratedSolutions/` - Images for Integrated Solutions service page

### Industry Pages
- `industries/healthcare/` - Images for Healthcare industry page
- `industries/ICT/` - Images for ICT industry page
- `industries/building&construction/` - Images for Building & Construction industry page
- `industries/financialServices/` - Images for Financial Services industry page
- `industries/transportation/` - Images for Transportation industry page
- `industries/consumerRetail/` - Images for Consumer Retail industry page

### Banner Images
- `banners/home/` - Banner images for the home page
- `banners/services/` - Banner images for the services page
- `banners/industries/` - Banner images for the industries page
- `consultancyBannerImages/` - Existing banner images folder (can be moved to banners/ if needed)

### Shared/Common Images
- `common/logos/` - Logo images (company logos, brand logos, etc.)
- `common/services/` - Service icons/images used across service pages
- `common/industries/` - Industry icons/images used across industry pages
- `common/icons/` - General icons and icon assets

## Notes
- Please organize images into their respective folders
- Keep common/shared images in the `common/` or `banners/` folders
- Update image paths in HTML files after moving images to their new locations
